# formula - A collection of formula functions

## Description

This module contains several formula functions that can be used in challenges.

## Functions

- **cube\_surface\_area(**z**)**

  Calculate the surface area of a cube with edge length `z`.

- **cube\_volume(**z**)**

  Calculate the volume of a cube with edge length `z`.

- **sphere\_surface\_area(**r**)**

  Calculate the surface area of a sphere with radius `r`.

- **sphere\_volume(**r**)**

  Calculate the volume of a sphere with radius `r`.
